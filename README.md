# boardgame-2p
A term project of AI course

## How to Compile and Run
- Python version: 3.7.4 up<br>
  Be sure of installing dependencies for all packages:
    - numpy==1.18.3
- Quick start<br>
  ```
  $ python game.py
  ```

## Search Alogrithm
Iterative deepening minimax with alpha-beta pruning
(timelimit = 15 sec)
